# 工程简介
Spring Cloud Alibaba 集成项目
集成 ~~nacos~~、mybatis plus、~~sentinel~~、redis 、SpringSecurity
目前进行搭建一个简单框架，

| 模块名称              | 模块 描述            |
|-------------------|------------------|
| ali-cloud-common  | 公共模块，中间件基础配置，工具类 |
| ali-cloud-dao     | 持久层mybaitis-plus |
| ali-cloud-service | 业务服务层，对web层提供服务  |
| ali-cloud-web     | web层,服务入口        |
